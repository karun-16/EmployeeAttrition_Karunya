# Employee Attrition Analysis

## Project Overview

This project analyzes employee attrition data to identify the factors that contribute to employees leaving a company. The objective is to understand attrition patterns, build predictive models, and provide business recommendations that can help HR teams improve employee retention.

## Dataset

Dataset: IBM HR Employee Attrition Dataset

The dataset contains employee information such as:

* Age
* Department
* Job Role
* Monthly Income
* Years at Company
* Work-Life Balance
* Overtime
* Stock Option Level
* Attrition Status

## Project Tasks

### Task 1 – Data Loading and Exploration

* Loaded and inspected the dataset
* Checked dataset structure and data types
* Generated descriptive statistics

### Task 2 – Data Cleaning and Preprocessing

* Handled categorical variables
* Encoded features for machine learning
* Prepared training and testing datasets

### Task 3 – Exploratory Data Analysis

* Analyzed attrition trends across departments and job roles
* Studied relationships between employee characteristics and attrition

### Task 4 – Model Building

Built and evaluated:

* Logistic Regression
* Random Forest
* Gradient Boosting

### Task 5 – Model Evaluation

Compared models using:

* Accuracy
* Precision
* Recall
* F1 Score
* ROC-AUC

Best Performing Model:
Gradient Boosting

### Task 6 – Visualization

Created:

1. Attrition Rate by Department and Job Role
2. Monthly Income Box Plot by Attrition Status
3. Confusion Matrix Heatmap
4. Top 10 Feature Importance Chart
5. ROC Curve Comparison

### Task 7 – HR Insights and Recommendations

Generated business-focused insights and recommendations for improving employee retention.

## Key Findings

* Employees working overtime showed a higher likelihood of leaving.
* Monthly income influences attrition, but it is not the only factor.
* Total working years and age also play important roles.
* Sales Representatives experienced the highest attrition among job roles.
* The Sales department should receive additional retention focus.
